# YU HWAPYEONG Project Hub

회계·세무 실무와 데이터 자동화 프로젝트를 한곳에서 보여주는 개인 포트폴리오 대시보드입니다.

## 주요 구성

- 실제 이력서를 반영한 소개, 기술 스택, 경력 및 주요 성과
- TalkToData와 6개의 GitHub Pages 프로젝트 링크
- 데스크톱과 모바일에 대응하는 반응형 레이아웃
- 영문 이력서 PDF 다운로드
- Open Graph 및 소셜 공유 이미지
